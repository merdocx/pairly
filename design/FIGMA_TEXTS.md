# Тексты интерфейса — синхронизация с Figma

## Автоматическая выгрузка из Figma

Скрипт `design/fetch-figma.mjs` загружает файл дизайна через Figma API и выгружает все тексты и цвета.

**Токен:** Personal Access Token из Figma (Account → Settings → Personal access tokens). Не коммитить в репозиторий.

**Запуск:**
```bash
cd /root/pairly
FIGMA_ACCESS_TOKEN=ваш_токен node design/fetch-figma.mjs
```
Либо добавьте в `.env`: `FIGMA_ACCESS_TOKEN=...` и запускайте так:
```bash
export $(grep -v '^#' .env | xargs) && node design/fetch-figma.mjs
```

**Результат:**
- `design/figma-texts.json` — все текстовые слои (path + text + style)
- `design/figma-export.json` — полный документ + тексты + уникальные цвета

После обновления макета в Figma запустите скрипт заново, затем попросите подставить новые тексты в код.

После обновления дизайна в Figma скопируйте сюда **все тексты** с экранов. Ниже — текущие тексты в коде для сверки.

## Тексты из Figma (обновлённые)

*Вставьте сюда тексты из обновлённого макета. Формат: экран → элемент → текст.*

Пример:
```
Логин:
- Заголовок: Pairly
- Подзаголовок: Смотрите фильмы вместе
- Кнопка: Войти
- Ссылка: Нет аккаунта? Регистрация
...
```
