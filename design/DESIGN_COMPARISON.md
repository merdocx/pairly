# Сравнение реализации с дизайном (Figma + PNG из root/design)

Референсы: Figma [Untitled](https://www.figma.com/design/OeztMeyJkgc6hbFlWNBRSm/Untitled), PNG в `root/design`.

**Полная автоматическая сверка (цвета, размеры, типографика, иконки):**  
`FIGMA_ACCESS_TOKEN=xxx node design/fetch-figma.mjs` — выгрузка из Figma и запуск `compare-figma.mjs`. Отчёт в консоли (✅/❌). Без токена: `node design/compare-figma.mjs` по уже выгруженным `figma-spec.json` / `figma-export.json`.

## Итог

- **Реализовано по макету:** цвета, градиент хедера, типографика, логин/регистрация (показ/скрытие пароля), порядок полей и подсказка пароля, заголовок и табы «Что смотреть», нижнее меню; поиск (лупа в поле, сетка 2 колонки, «+ Добавить»); главная (сетка карточек, «Отметить просмотренным», звёзды); профиль (карточка пользователя, модалка «Добавить пару» с кодом и копированием, статистика, красная кнопка «Выйти»); деталь фильма (X закрыть, зелёная галочка в списке, звёзды 1–5, тег «Добавлено вами», зелёная «Просмотрено»); модалка «Оценить» со звёздами, «Удалить оценку», «Отмена».
- Все пункты из чеклиста приведены в соответствие с дизайном.
